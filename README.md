## 🎯 Sobre o Projeto

Este projeto é um jogo desenvolvido em Python que aplica conceitos fundamentais de **Programação Orientada a Objetos** (POO), como:
- Classes e Objetos
- Herança
- Encapsulamento
- Polimorfismo

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- Git para controle de versão
- GitHub para hospedagem do repositório

## ⚙️ Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

- [Python 3.8+](https://www.python.org/downloads/)
- Git (opcional, para clonar o repositório)

## 🚀 Como Executar

```bash
# Clone este repositório
git clone git@github.com:AugustoFFG/Game.git

# Acesse a pasta do projeto no terminal
cd Game

# Execute o jogo
python main.py
